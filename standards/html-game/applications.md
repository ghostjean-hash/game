# 2. 적용 절차 체크리스트 (본 표준 9.1)

각 프로젝트 적용 시 아래 항목 모두 충족.

2.1. STANDARD.md 6장 순서대로 셋업 완료.
2.2. 게임 루트에 `.standard` 파일 (한 줄: `html-game v<x.y>`).
2.3. CLAUDE.md 8장 변경 이력 첫 줄에 `YYYY-MM-DD: html-game v<x.y> 적용` 기재.
2.4. 본 파일 1번 표에 행 추가.

# 3. 마이그레이션 메모

3.1. 표준 버전이 올라가도 적용 프로젝트는 자동 갱신되지 않습니다.
3.2. 표준 변경 시 본 파일 1번 표의 적용 버전과 STANDARD.md 현재 버전 비교 후, 마이그레이션 필요 여부 사용자 결정.
3.3. 마이그레이션 결정 시 해당 프로젝트의 `.standard` 파일과 CLAUDE.md 변경 이력 갱신.
