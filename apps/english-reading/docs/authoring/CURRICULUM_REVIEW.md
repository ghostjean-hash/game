# 영어 독해 앱 200편 커리큘럼 - 확정본 v3 (ChatGPT·Gemini·Claude 3차 감수 반영)

> 200편의 주제 설계(무엇을 어느 난이도로 다룰지). 세 차례 감수(ChatGPT·Gemini 2회 + Claude 재감수)를 반영해 63건을 수정한 확정본이다.
> 작성: Claude Code / 감수: ChatGPT, Gemini, Claude

## 1. 이 앱이 무엇인가

- 다독(흐름 읽기)과 정독(끊어읽기·단어·구조 해설)을 한 화면에 합친 하이브리드 리딩 학습 도구.
- 학습자는 지문을 몰입해 읽다가, 막히는 문장에서만 손끝으로 끊어읽기(/)·단어 뜻·구조 해설을 꺼내 쓴다.
- 타이핑·강제 문제 풀이·정답 채점이 없다. 100% 클릭·터치. 흐름을 우선하는 철학.
- 최종 학습 목표(단계): 정확한 독해 → 영어 어순 이해 → 듣기 → 간단한 말하기 → 쉬운 영어 영상 이해.

## 2. 확정된 설계 기준

- 총량: **지문 200편 × 지문당 5문장 = 1,000문장**.
- 난이도: **3단계(Level 1 / 2 / 3)**. 목표 수준 **한국 중학교 중심 ~ 고1 초입**. 수능·학술·성인 고급은 범위 밖.
  - Level 1: 대체로 7~11단어. 단순 현재·과거, 기본 조동사. 핵심 구조 1개. (초등 고학년~중1)
  - Level 2: 대체로 8~14단어. that절, to부정사, 동명사, 짧은 관계절, 간접의문문. (중2~중3)
  - Level 3: 대체로 9~18단어. 관계절, 비교절, 수동태, 분사 수식. "쉬운 설명 영상" 수준. (중3~고1 초입)
- 난이도 분포: **Level 1 = 60편 / Level 2 = 80편 / Level 3 = 60편**.
- 숙어·구동사 정책: 평균 지문당 1개, 쉬운 단계일수록 적게. Level 1은 대부분 없음, Level 2·3은 문맥이 맞을 때만. 현재 확정본 숙어 슬롯 97개, 고유 숙어 89종(같은 숙어 3회 이상 반복 0).

## 4. 감수 반영 이력 (v1 → v3)

세 차례 감수의 타당한 지적을 반영해 63건을 수정했다. 배분(60/80/60)과 총 200편은 그대로 유지.

- **중복 소재 교체**: 기존 25편·비교 출제분과 겹치거나 topic 안에서 전개가 반복되던 소재를 서사가 살아나는 소재로 교체(예: 10 달팽이 발견, 29 혼자 있는 새 학생, 52 바뀐 탑승구, 148 SNS 비교, 196 아이스크림·물놀이 상관관계 등).
- **숙어 과다중복 해소**: 같은 숙어 반복(figure out·put off·build up·cheer up·pick up)을 분산·삭제. 3회 이상 반복 숙어 0으로 정리.
- **숙어·소재 부적합 교정**: 문맥과 안 맞는 숙어 교체(open up→air out, keep to oneself→keep a secret, keep at→keep at it) 또는 억지 숙어 삭제(달의 위상 make out, 하늘색 made up of 등).
- **난이도·안전 조정**: 건강 오해 소지(주말 몰아자기) 완화, Level 3 과도 난도(번역 원리→어순 차이 초점) 완화.
- **흥미 개선**: 뻔한 설명형을 체험형으로(135 다크 모드 장단점 체험 등).
- **교차 topic 중복·정확성(3차 감수)**: topic을 가로지르는 주제 군집을 차별화(78 수면의 질, 109 과거의 나와 비교), 느슨한 숙어 라벨 교체(128 come to mean, 190 trap heat), 과학 오개념 방지(179 밀도 less dense), 소재 무게·중복 조정(196 수영장 방문객, 162 카드게임).

미채택(선택 남김): 두 감수의 '추가 제안 소재'(숏폼 알고리즘, 이스포츠, 디지털 디톡스 등)는 200편 정원이 차 있어 반영하지 않았다. 원하면 기존 소재와 맞교체 가능.

## 6. 다음 단계

이 확정본 지도를 기준으로 실제 5문장 지문 원문을 배치(5~10편)로 출제한다. 각 배치는 출제 직후 앱 검증기(`validatePassage` + `compareAgainstExisting` + 끊어읽기 규칙)로 검사하고, 통과분만 `passages.json`에 반영한다.

### 원문 배치 규칙 (3차 감수 반영)

- **시간대 분산**: '아침·기상' 배경이 특정 구간에 몰리지 않게 점심·저녁·밤·주말을 의도적으로 섞는다.
- **문장 길이 리듬**: Level 1은 5문장을 7~11단어로 들쭉날쭉 섞는다(모두 같은 길이 금지). Level 3은 18단어에 육박하는 긴 문장을 지문당 1~2개로 제한한다.
- **숙어의 자연스러운 스며듦**: 억지 삽입 금지. 구동사는 목적어 위치(clean it up / clean up the room)가 끊어읽기 규칙과 충돌하지 않게 배치한다.
- **대명사 설계**: it·they·this가 앞 문장의 무엇을 가리키는지 직관적으로 인지되도록 문장 간 인과를 명확히 한다.
- **주제 프레임 명시**: 인접 주제(미루기의 결과 vs 심리기제 등)는 원문에서 초점을 분명히 갈라 중복 체감을 줄인다.
