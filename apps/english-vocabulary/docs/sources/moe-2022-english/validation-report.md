# 2022 개정 영어과 별책14 어휘 - 범례 확정 + 추출 재검산

## 1. 별표 범례 확인 (공식 원문 확보)

범례는 임베디드 이미지가 아니라 `[별표 3] 어휘` 앞의 표(table) 텍스트였다. HWP `BodyText/Section0` 레코드를 직접 파싱해 원문을 확보했다(추론 아님).

원문 인용 (기본 어휘 관련 지침 1항):
> "영어과 기본 어휘 목록은 ... 가장 기본적인 3,000개의 단어로 되어 있다. *표를 한 단어는 800개로 초등학교에서 사용하기를 권장하고, **표를 한 단어는 1,200개로 중학교와 고등학교 공통과목에서 사용하기를 권장한다. 나머지 1,000개의 단어는 그 외의 과목에서 사용하기를 권장한다."

### 별표 → 의미 확정

| 표기 | 개수(공식) | 의미(공식) |
|---|---|---|
| `*` (홑별표) | 800 | 초등학교에서 사용 권장 |
| `**` (겹별표) | 1,200 | 중학교와 고등학교 공통과목에서 사용 권장 |
| 무표시 | 1,000 | 그 외의 과목에서 사용 권장 |
| 합계 | 3,000 | 기본 어휘 목록 |

- 학교급 단위(학년군 아님). "사용 권장"이며 `[별표4]`의 "학습 시작 권장 학교급(●)"과는 다른 표기 체계다.
- 별표는 개별 용례가 아니라 **대표형(단어군)** 단위로 부여된다(지침 4항).

### 추가 공식 규정 (앱 데이터 설계에 중요)

- 지침 4: 목록은 **대표형**으로 제시. 대표형은 그 자체 + 모든 굴절 변이형 + 36개 파생 접사 변이형을 대표. 고빈도 파생형 일부는 대표형 옆 `( )`에 표기(예: `able** (disabled)`).
- 지침 5: 고유명사, 호칭, 감탄사, 로마자화 외국어, 알파벳/문자, 단위명, 약어, 화학식, (일부 제외한) 기수·서수는 기본 어휘로도 새 어휘로도 간주하지 않음. 단 first/second/third/twenty-first/second/third 등 지정 서수는 목록에 포함됨.
- 지침 2: 학년군별 학습 어휘 수 - 초3~4 300 이내, 초5~6 300 이내(누계 600), 중1~3 1,500 이내, 고교 공통/선택은 과목별 1,600~3,500.

## 2. 3,045 추출 재검산

### 개수 정합 (완전 일치)

| 구분 | 대표형(entry) | 슬래시 분리(word) |
|---|---|---|
| `*` | 800 | 819 |
| `**` | 1,200 | 1,215 |
| 무표시 | 1,000 | 1,011 |
| 합계 | **3,000** | **3,045** |

- 대표형 3,000 / 800 / 1,200 / 1,000 = **공식 범례 수치와 완전히 일치**.
- 앞서 보고한 3,045는 슬래시 이형(예 `yes*/yeah*/yep*`, `color*/colour*`) 40개 항목을 개별 철자로 분리해 +45된 값 = 3,000 대표형 + 45. 수치가 정확히 설명된다.
- 두 독립 추출 방법(hwp5txt 툴 vs OLE 레코드 직접 파싱)의 단어 집합이 **완전 동일**(diff 0) - 상호 교차검증 통과.

### 발견 오류 및 수정

1. 줄바꿈 분리 1건: 원문 `administer (administration)`의 괄호 조각이 다음 줄로 넘어가 `(administration)`가 별도 항목(무표시)으로 잡혔다 → 이전 초기 집계가 무표시 1,001/총 3,001로 +1 과다. 괄호 조각(`(`로 시작하는 줄) 병합 규칙 추가로 수정 → 무표시 1,000/총 3,000.
2. 국지적 정렬 이상 2건(`administer/(administration)`, `mode/modify/model/moderate/modern`): 원문 레코드 저장 순서가 다단 표 구조라 시각 알파벳 순과 국소적으로 어긋남. **단어 자체는 모두 온전**(분리·병합·손상 없음). 데이터 무결성 문제 아님.
3. 하이픈 단어: `twenty-first/second/third` 3건 - 지침 5의 "포함되는 지정 서수"로 정상 항목.
4. 괄호 주석이 표제어로 오포함된 사례: 0건(수정 후). 중복 표제어: 0건. 이상 토큰(비알파벳): 0건.
5. 알파벳 첫 단어 `a`, 마지막 `zoo`까지 A~Z 전 구간 누락 없음. 섹션 구분 대문자(A,B,C...)는 단어 아님으로 제외(X는 표제어 없어 25개).

### 표본 대조 (각 그룹 앞·중간·끝 30개, 원문 대조 OK)

- `*` (초등): a, about, above, across, add, address, adult ... key, kick, kid ... write, wrong, year, yellow, yes/yeah/yep, yesterday, you, young, zoo
- `**` (중고 공통): able, absolute, accent, accept, access ... lift, limit, link, list, load ... wrap, yell, yet, zebra, zero
- 무표시 (그 외): abandon, aboard, abort, abound, abroad, absent ... inhabit, inherent, inhibit ... withdraw, witness, worship, wreck, yield, zone
