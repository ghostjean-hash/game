# 교육부 2022 개정 영어과 기본 어휘 목록 (공식 원천)

이 폴더는 앱 어휘의 **단일 공식 원천 후보**다. 가공 데이터(set-NNN.json)와 분리해 원천만 보존한다. 검증 절차는 `../../vocab-master-plan.md` 2장 규칙을 따른다.

## 1. 출처 (provenance)

- 문서: 「2022 개정 영어과 교육과정」 별책 14
- 고시: 교육부 고시 제2022-33호
- 원본: `byeolchaek14_english.hwp` (HWP 5.x, moe.go.kr 별책5~14 묶음 ZIP 15.1MB 내). 다운로드 URL은 `metadata.json` 참조.
- 확보 방식: HWP를 hwp5txt + OLE BodyText 레코드 직접 파싱(두 경로 교차검증, 단어 집합 diff 0).

## 2. 별표(범례) - 공식 원문 확정 (markerMappingConfirmed: true)

「기본 어휘 관련 지침」 1항 원문 기준(전문은 `legend-and-guidelines.txt`):

| 표기 | 대표형 개수 | 의미 |
|---|---|---|
| `*` | 800 | 초등학교에서 사용 권장 |
| `**` | 1,200 | **중학교와 고등학교 공통과목**에서 사용 권장 |
| 무표시 | 1,000 | 그 외의 과목에서 사용 권장 |

**주의(중요)**: 별표는 "초/중/고 각각"이 아니다. `**`는 중학과 고등에 **공통**이고, 무표시는 선택·심화 과목용이다. 따라서 별표만으로는 "중학 전용 vs 고등 전용"이 갈리지 않는다. 학년군별 어휘 수 상한은 지침 2항 참조(초등 누계 600 / 중학 1,500 / 고등 공통영어 1,800 … 심화 3,000~3,500 이내).

또한 별표는 **대표형(단어군) 단위**로 부여된다(지침 4항: 굴절형 + 36개 파생접사 변이형 대표). 고유명사·호칭·감탄사·단위·약어·화학식·일부 기수/서수는 어휘로 미계수(지침 5항).

## 4. 검산 (독립 재확인)

대표형 3,000 = `*` 800 + `**` 1,200 + 무표시 1,000. 공식 지침 1항 수치와 완전 일치(두 독립 추출 교차검증 + 별도 재검산). 슬래시 분리 시 3,045(+45).
