# Sky Raider

좌→우 전진 캐주얼 횡스크롤 비행 슈팅. 화면을 손가락으로 끌어 기체를 움직이고, 총알은 자동으로 나간다. 3개 구역의 보스를 격파하면 클리어.

- **조작**: 드래그(이동) + 자동발사. PC는 방향키/WASD.
- **성장**: 적을 잡으면 나오는 P(화력)·♥(회복)·B(봄) 아이템.
- **목숨**: 3개. 부딪히면 하나 줄고 화력도 한 단계 떨어진다.

바닐라 JS + Canvas. 이미지/오디오 파일 없이 도형과 Web Audio 합성으로만 구성(자산 0). 게임 허브(game_ghost) 공유 자산(`shared/`)을 사용한다.
