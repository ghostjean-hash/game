# Sky Raider

세로(종) 스크롤 캐주얼 비행 슈팅. 화면을 손가락으로 끌어 기체를 움직이고(하단), 위에서 내려오는 적을 향해 총알은 자동으로 나간다. 10개 구역의 보스를 격파하면 클리어.

- **조작**: 드래그(이동) + 자동발사. PC는 방향키/WASD.
- **성장**: 세 갈래 파워 파츠를 전용 아이템으로 키운다. P(앞쪽 화력 최대 8발)·S(좌우 옵션 비행기 최대 8대, 안쪽 레이저·바깥 유도미사일)·E(주변 에너지존, 닿는 적에 지속 피해). 그밖에 ♥(회복)·B(봄).
- **목숨**: 3개. 부딪히면 하나 줄고 마지막에 얻은 파츠도 하나 사라진다.

바닐라 JS + Canvas. 이미지/오디오 파일 없이 도형과 Web Audio 합성으로만 구성(자산 0). 게임 허브(game_ghost) 공유 자산(`shared/`)을 사용한다.
