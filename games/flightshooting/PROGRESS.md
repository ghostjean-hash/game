### 완료 - 자동 플레이 (core/autopilot.js 3차 재설계)
- 최종 방식은 '짧은 미래 시뮬레이션(rollout)'. 갈 수 있는 여러 목표 지점마다 실제로 이동하며 모든 적탄·적기·보스를 실제 궤적대로 SIM_T(1.2초)까지 굴려 '언제 죽는가(생존 시간)'를 재고, 가장 오래 사는 목표를 고른다. 안 죽는 목표가 여럿이면 조준·드랍 획득 이득으로 tie-break해 회피·조준·획득을 한 점수로 통합.
- 후보에 세로·대각 이동 포함(가로 전용 탈피, 위쪽 드랍 마중·틈새 회피). 목숨 최대면 회복 하트(H) 무시.
- 보너스 기체를 최우선 조준 + 예측 조준(lead: 총알 도달 시간만큼 표적 가로 이동 미리 겨냥)으로 희소 드롭에도 최대 성장 도달.

## 2026-07-07 (후속4) - 파워 파츠 3계통 강화 + 마스터 표시

### 사용자 지시 (전수)
- P를 업그레이드하면 색이 점점 붉은 계열로 / S(보조 비행기)에 파츠 추가되면 발사체 크기 커지게 / E(주변)는 색 변경 + 적 총알을 조금씩 밀어내는 기능 / 그 외 더 강화할 것 체크.
- (후속) P가 8개로 분화하는 동안 탄 크기가 너무 커지지 않게 완만히 / 모든 Max 상태를 '마스터' 개념으로 최대 강화 단계임을 표현.

### 완료 - 3계통 강화
- **P 전방 화력**: 단계별 탄 색 = colors.bulletByLevel(1 청록 → 8 붉은색). view.drawBullets가 game.front로 색 선택. 탄 굵기 공식은 fire.js `3 + dmg*0.8` → `CFG.parts.front.rBase(2.8) + dmg*rGrow(0.42)`로 완만화(8단계 반경 6.2→4.48).
- **S 옵션기**: 발사체 크기를 옵션 수에 비례(laserR/missileR base + 옵션수 × grow). parts.stepOptions가 game.options.length로 계산. 8대면 미사일 반경 4→7.6.
- **E 에너지존**: 레벨별 색 = colors.zoneRgbByLevel(보라 → 라임, rgb 문자열 + drawZone이 alpha 조합). 신규 repelEnemyBullets(parts.js) - 존 반경 안 적탄을 플레이어 바깥으로 민다(안쪽·고레벨일수록 강, pushBase 40 + level×48). world.stepWorld가 매 프레임 호출(완전 차단 아닌 밀어내기).
