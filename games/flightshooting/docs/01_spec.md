# 01 게임 사양 (Sky Raider)

## 1. 한 줄

세로(종) 스크롤 캐주얼 비행 슈팅. 플레이어는 화면 하단, 적은 위에서 내려온다. 10개 구역의 보스를 격파하면 클리어.

## 2. 조작

- **이동**: 화면 드래그(상대 이동 - 잡은 지점 대비 델타를 기체에 적용, 손가락 가림 방지). PC는 방향키/WASD.
- **발사**: 자동(0.14초 간격). 정면(위쪽) 기준 화력 단계별 부채.
- **일시정지**: ⏸ 버튼 또는 P 키. 일시정지 중 화면 탭으로 재개.
- **전체화면**: 지원 기기에서 ⛶ 버튼(미지원 시 숨김).

## 3. 코어 루프

웨이브(적 무리)를 처치하며 전진 → 웨이브 소진 후 보스 등장 → 보스 격파 시 다음 구역. 목숨 3개가 모두 소진되면 게임오버.

## 4. 화력 성장 (1~20단계)

- 파워업 P 획득 시 +1, 피격 시 -1(최소 1).
- 홀수 단계 = 탄 갈래 추가(최대 10발), 짝수 단계 = 데미지 + 탄 굵기 상승. 매 단계 반드시 체감된다.
- 12단계에서 측면탄, 17단계에서 후방탄 추가.
- 계산식은 `src/core/fire.js`의 `fireSpec(L)`. 수치는 → 02_data.md.

## 5. 적

- **drone**: 직진 하강. hp 1.
- **weaver**: 좌우 사인파로 흔들며 하강. hp 2.
- **gunner**: 느리게 하강하며 플레이어 x를 추적, 조준 발사. hp 3.
- 상세 수치 → 02_data.md.

## 6. 보스

- **중보스(1~9구역)**: 작은 기체. 좌우 유영 + 조준 발사 + 호위 비행체 주기 소환. hp는 구역에 비례.
- **최종보스(10구역)**: 큰 기체. 좌우 유영 + 2패턴(아래 방향 부채 산탄 / 조준 3연발) 번갈아. 고정 hp.

## 7. 파워업 아이템 (적 처치 시 확률 드롭)

- **P**: 화력 +1.
- **H**: 목숨 +1(최대 3).
- **B**: 봄 - 화면의 적 전멸 + 보스에 큰 데미지.

## 8. 화면 흐름

메뉴 → (시작) → 구역 1 웨이브 → 중보스 → 구역 2 … → 구역 10 최종보스 → 전 구역 격파(승리). 중간 목숨 소진 시 게임오버 모달(다시하기/홈).

## 9. 성공 / 실패 / 종료

- 성공: 10구역 최종보스 격파.
- 실패: 목숨 0.
- 진행 유지: 화력·목숨·점수는 구역이 넘어가도 유지(초기화는 게임 재시작 시만).
- 점수: 적/보스 처치 누적. localStorage 최고 기록(`gg.flightshooting.best`).
