# 01 게임 사양 (Sky Raider)

## 1. 한 줄

세로(종) 스크롤 캐주얼 비행 슈팅. 플레이어는 화면 하단, 적은 위에서 내려온다. 10개 구역의 보스를 격파하면 클리어.

## 2. 조작

- **이동**: 화면 드래그(상대 이동 - 잡은 지점 대비 델타를 기체에 적용, 손가락 가림 방지). PC는 방향키/WASD.
- **발사**: 자동(0.14초 간격). 정면(위쪽) 기준 화력 단계별 부채.
- **일시정지**: ⏸ 버튼 또는 P 키. 일시정지 중 화면 탭으로 재개.
- **전체화면**: 지원 기기에서 ⛶ 버튼(미지원 시 숨김).
- **자동 플레이**: 🤖 버튼으로 켜고 끈다. 기체가 스스로 적·적탄을 피하고 파워업을 주우며 진행한다(발사는 자동). 데모·밸런스 체감·검증용. 로직은 `core/autopilot.js`.

## 3. 코어 루프

웨이브(적 무리)를 처치하며 전진 → 웨이브 소진 후 보스 등장 → 보스 격파 시 다음 구역. 목숨 3개가 모두 소진되면 게임오버. 구역이 바뀌면 '구역 N' 배너가 잠시(`stageIntro`) 뜨고 그동안 적이 나오지 않는다(전환 숨돌림).

## 4. 파워 파츠 (3계통)

파워업이 세 갈래로 분리된다. 각 계통은 전용 아이템으로만 성장하고, 피격 시 마지막 얻은 파츠 1개를 잃는다(역순 손실). 상세 설계 → 05_power-parts.md.

- **전방 화력(P)**: 정면 부채탄. 1~8단계, 단계당 탄 +1(최대 8발), 데미지 완만 상승. `core/fire.js`의 `frontSpec(L)`.
- **옵션기(S)**: 좌우에 붙는 부속 비행기. S 획득당 좌우 번갈아 1대씩, 최대 8대(좌4·우4). 안쪽 2슬롯 레이저(빠른 직진탄), 바깥 2슬롯 유도 미사일. `core/parts.js`.
- **에너지존(E)**: 플레이어 주변 원형 오라. 레벨 0~5, 레벨↑ 반경↑. 0.5초마다 존 안의 적·보스에 레벨만큼 피해. `core/parts.js`.

## 5. 적

- **drone**: 직진 하강. hp 1.
- **weaver**: 좌우 사인파로 흔들며 하강. hp 2.
- **gunner**: 느리게 하강하며 플레이어 x를 추적, 조준 발사. hp 3.
- 적 체력은 구역이 오를수록 상승: `base × (1 + (stage-1)×enemyHpScale)` (10구역 ≈ 3.5배). 3계통 화력 성장과 균형.
- 상세 수치 → 02_data.md.

## 6. 보스

- **중보스(1~9구역)**: 작은 기체. 좌우 유영 + 조준 발사 + 호위 비행체 주기 소환. hp는 구역에 비례.
- **최종보스(10구역)**: 큰 기체. 좌우 유영 + 2패턴(아래 방향 부채 산탄 / 조준 3연발) 번갈아. 고정 hp.

## 7. 파워업 아이템 (적 처치 시 확률 드롭, 5종)

- **P**: 전방 화력 +1 (최대 8).
- **S**: 옵션기 1대 추가 (최대 8).
- **E**: 에너지존 레벨 +1 (최대 5).
- **H**: 목숨 +1(최대 3).
- **B**: 봄 - 화면의 적 전멸 + 보스에 큰 데미지.
- 피격 시: 목숨 -1 + 마지막 얻은 파츠(P/S/E) 1개 손실.

## 8. 화면 흐름

메뉴 → (시작) → 구역 1 웨이브 → 중보스 → 구역 2 … → 구역 10 최종보스 → 전 구역 격파(승리). 중간 목숨 소진 시 게임오버 모달(다시하기/홈).

## 9. 성공 / 실패 / 종료

- 성공: 10구역 최종보스 격파.
- 실패: 목숨 0.
- 진행 유지: 파워 파츠·목숨·점수는 구역이 넘어가도 유지(초기화는 게임 재시작 시만).
- 점수: 적/보스 처치 누적. localStorage 최고 기록(`gg.flightshooting.best`).
