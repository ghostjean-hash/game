# 01 게임 사양 (바푸리의 모험)

## 1. 한 줄

세로(종) 스크롤 캐주얼 비행 슈팅. 플레이어는 화면 하단, 적은 위에서 내려온다. 40개 구역의 보스를 격파하면 클리어(11구역부터 신규 적, 21구역부터 완전 다른 형태의 이질 기계 적 - 상세 docs/08, 31구역부터 빛·에너지 생명체 - 상세 docs/12).

## 2. 조작

- **난이도(시작 화면 선택)**: 홈에서 **쉬움 / 보통 / 어려움 / 매우 어려움** 4단계 중 고른다(`CFG.difficulty`, 세로 목록 + 선택 항목에 체크마크). 난이도는 세 가지를 함께 조절한다 - 적·보스가 쏘는 총알 양(`enemyFireMul`), 적 체력 배수(`enemyHpMul`), 내 목숨 수(`maxLives`, 쉬움 5 / 보통 3 / 어려움 2 / 매우 어려움 1). '쉬움'은 옛 어린이 배려를 흡수한 가장 편한 난이도로, 적 조준을 정중앙 단발로 줄이고(`enemyShotsMax`), 이질 적의 방사·자폭 탄(기뢰 파편·결정체 반사)을 감축하며(`radialMul`), 메인 총알 2발·꼬리 비행기 1대를 갖춘 채 출발한다(`startFront`/`startTail`). **보통**은 적 발사 간격·체력을 아주 조금 완화하고, 1~10구역의 조준 연발을 최대 2발로 제한한다. 선택은 localStorage에 기억된다. (2026-07-16 개편으로 옛 '일반/어린이 모드'는 폐지 - 어린이 배려는 '쉬움'에, 친구는 아래 토글로 분리됐다.)
- **친구 동행 / 자동 플레이(홈 아이콘 토글)**: 난이도와 별개로 홈에서 아이콘 on/off로 켠다(색으로 구분 - 난이도 시안 / 친구 초록 / 자동 보라). **친구 동행**은 함께 싸우는 동료 키위새(상세 09_friend.md)를 부르고, **자동 플레이**는 시작부터 자동 조종을 켠다. 둘 다 어느 난이도와도 자유롭게 조합한다.
- **이동**: 화면 드래그(상대 이동 - 잡은 지점 대비 델타를 기체에 적용, 손가락 가림 방지). PC는 방향키/WASD.
- **발사**: 자동(약 0.15초 간격, `player.fireEvery`). 메인 총알은 부채 없이 나란히 곧게 직진하고, 부채 확산은 사이드 총알(옵션기)이 맡는다.
- **일시정지**: ⏸ 버튼 또는 P 키. 일시정지 중 화면 탭으로 재개.
- **전체화면**: 지원 기기에서 ⛶ 버튼(미지원 시 숨김).
- **자동 플레이(하이브리드)**: 홈의 자동 플레이 토글 또는 게임 중 🤖 버튼으로 켠다. 켜져 있어도 **내가 화면을 조작하면 즉시 내 조작이 우선**하고, **손을 떼면 0.5초 뒤 자동으로 복귀**한다(`autoAssist`+`manualTimer`/`dragging`, `resumeDelay` 0.5). 자동일 때 기체가 스스로 적·적탄을 피하고 파워업을 주우며 진행한다(발사는 자동). 로직 `core/autopilot.js`. **실력 4단계**(초보/중수/고수/프로, 환경설정에서 선택, **기본 프로**) - 반응 지연·조준 오차·예측 지평·동시 위협 수를 인간 실측 기준으로 조절해 사람처럼 움직인다(근거 docs/plans/research-2026-07-09).
- **점수**: 직접 조작 중에는 처치·보너스·보스·최대 강화 보너스의 원래 점수를 얻는다. 자동 플레이가 실제로 조종하는 동안에는 같은 획득 점수의 1/3만 얻는다. 자동 플레이를 켜도 손으로 조작 중이면 원래 점수다.
- **환경설정**: 홈 '환경설정'에서 **조작법 안내**(옛 홈의 '조작법' 버튼을 이 모달로 이관) + 자동 플레이 실력 선택 + 치트 박스 on/off를 본다. 치트 박스(드래그 이동·접기)는 플레이 속도(×1~×8)·무적·아이템 드랍 확률·종류를 조절한다(개발·테스트용). 치트 세부 설정은 localStorage(`cheatCfg`)에 저장돼 재시작·재접속에도 유지되고, 치트 박스가 열릴 때 `syncCheatUI`로 저장값이 UI에 반영된다(사용자 지시 2026-07-10).

## 2.5 캐릭터·표정 (2026-07-16)

- 플레이어는 **바푸리**: 둥근 흰 몸통 + 몸에 붙은 손발 + 붉은 'O' 입 + 작은 눈. 몸은 위(진행 방향)로 나아가되 얼굴을 몸 하단부에 둬 고개를 돌려 플레이어를 보는 느낌(`view.drawPlayer` + 홈 대표 SVG).
- **표정**(`view.drawFace`, 바푸리·친구 키위새 공통): 총알에 맞으면 우는 표정(감은 눈 + 파란 눈물 + 벌린 입), 아이템을 먹으면 웃는 표정(눈 ^^ + 웃는 입). 잠깐(`numbers.emote` cry/happy) 뒤 기본 표정으로 돌아온다. 상태는 `core/world.js`(피격·획득)·`core/friend.js`(친구 피격)가 `emo`/`emoT`로 세팅.

## 3. 코어 루프

웨이브(적 무리)를 처치하며 전진 → 웨이브 소진 후 보스 등장 → 보스 격파 시 다음 구역. 목숨(난이도별)이 모두 소진되면 **큰 폭발 연출** 뒤(`numbers.emote.deathTime` 약 1.1초) 게임오버 모달이 뜬다('dying' 상태 경유). 구역이 바뀌면 '구역 N' 배너가 잠시(`stageIntro`) 뜨고 그동안 적이 나오지 않는다(전환 숨돌림).

세계 여행에서는 보스 폭발의 주 연출이 1초간 진행되고, 마지막 폭발 파티클까지 사라진 뒤에만 월드맵을 연다. 파티클이 비정상적으로 남으면 3초에 해당 파티클을 정리하고 월드맵으로 전환한다. 도시를 고른 뒤에는 다음 도시의 디오라마 배경 이미지 로딩 완료(파일이 없으면 fallback 확정)까지 지도 화면을 유지하며, 그 전에는 다음 구역 전투를 시작하지 않는다.

## 4. 파워 파츠 (4계통)

파워업이 네 갈래로 분리된다. 각 계통은 전용 아이템으로만 성장하고, 피격 시 마지막 얻은 파츠 1개를 잃는다(역순 손실). 모든 발사체는 모양·크기가 커질수록 데미지가 오른다(에너지존만 예외). 상세 설계 → 05_power-parts.md.

- **강화는 발별 순차 진화 10단계**(사용자 확정): 강화 아이템 하나당 총알 한 발씩 순차로 진화한다(6발 일괄 아님). 총알마다 자기 tier 0(무강화)~10을 가지고, 진화한 총알만 형태·데미지·속도가 오른다. 모든 총알 속도는 강화 3단계마다 한 계단 빨라진다(`bullet.speedPer3` 0.15, 초기 느림).
- **전방 화력(P)**: 정면 직진 레이저 빔. `front` 1~66단계. 1~6단계는 탄 +1(최대 6발, 개별 탄 균일, 데미지 1,1,2,2,3,3). 7~66단계는 6발 고정, **가운데 탄부터 한 발씩 순차 진화**(6스텝마다 그 탄 tier +1, 최대 10). 빔 몸통 형태가 tier마다 패턴(실선→마디→구슬→마름모→물결→톱니→이중→나선→화살→플라즈마, `view.drawMainBeam`이 tier로 그림). 색은 아군 냉색(`mainTier`), tier↑ 데미지·속도·길이·굵기 상승. `core/fire.js`의 `frontSpec(front)`.
- **옵션기(S)**: 좌우 부속 비행기, 최대 6대(좌3·우3). **6대 전부 레이저**(부채 확산), 옵션 수↑ 굵기·데미지 상승. 6대 채운 뒤 **안쪽 총알부터 한 발씩 순차 진화**(발별 순차, 6스텝마다 tier +1, 최대 10). 둥근 형태(이슬→구슬→방울→고리→별→태양, `view.drawSideShape`). `core/parts.js`.
- **에너지존(E)**: 플레이어 중심에서 퍼지는 펄스파(파동 링). 레벨 0~5. 링이 주기적으로 바깥으로 퍼지고, **링이 지나가는 순간 링 위에 있는 적·보스에게만** 레벨만큼 피해(상시 장판 아님). 레벨↑ 파동이 더 자주·멀리·두껍게·세게. `core/parts.js`.
- **꼬리 비행기(T)**: 플레이어 뒤 유도탄 발사기, 최대 4대(처음부터 유도탄). 대수(1~4대) 채운 뒤 **앞(낮은 단계) 비행기부터 한 대씩 순차로 무기 진화**(발별 순차, `weapon` 1~11 = tier 0~10). 형태가 점→원→삼각→화살→로켓→미사일→다탄두로 진화(몸체 색 3색 순환 `tailMissile3`), 단계↑ 크기·데미지·속도 상승. 유도탄이 가장 가까운 적을 추적. `core/parts.js`.

## 5. 적

기본 적(1~10구역 주력):
- **drone**: 직진 하강. hp 1. 발사 없음.
- **weaver**: 좌우 사인파로 흔들며 하강. hp 2. 가끔 단발 조준 발사.
- **gunner**: 느리게 하강하며 플레이어 x를 추적, 3발 확산 조준 발사. hp 3.

신규 적(11~20구역 주력, "전혀 다른 적"):
- **splitter(분열체)**: 직하하다 격파되면 조각(shard)으로 쪼개져 좌우로 흩어진다. shard는 작고 빠르며 재분열하지 않는다.
- **shielder(방패병)**: 정면(아래)을 향한 방패로 기본탄·레이저를 막는다. 꼬리 비행기의 유도탄(측면 우회)이나 에너지존(범위)으로만 피해를 줄 수 있어 그쪽 성장을 유도. 단발 조준 발사.
- **rusher(돌격기)**: 잠시 조준하며 천천히 내려오다, 플레이어 방향으로 급강하 돌진한다.

이질 기계 적(21~30구역 주력, "완전 다른 형태" - 각진 기계·결정 계열, 상세 docs/08):
- **turret(부유 포대)**: 거의 정지한 채 포신으로 3방향 조준 연사. 단단하다.
- **prism(결정체)**: 직하하되 피격당할 때마다 사방으로 반사탄을 튕긴다.
- **mine(기뢰)**: 느리게 표류하다 플레이어가 근접하면 자폭(범위 파편탄) - 정면 돌파를 벌한다.
- **warper(공간 왜곡체)**: 짧게 순간이동하며 하강해 위치 예측이 어렵다.
- **coil(전격 코일, 26구역~)**: 두 노드가 전기 아크 선으로 연결된 쌍으로 나란히 내려온다. 두 노드 사이 아크 선에 닿으면 피해 - 한쪽 노드를 부수면 아크가 사라진다. 위치 잡기를 압박한다.
- **serpent(기계 뱀, 28구역~)**: 머리 + 몸통 마디가 이어진 체인. 머리만 약점이고 몸통은 아군탄을 막는다(무적). 사인파로 구불거리며 내려와 머리를 노려야 전체가 격파된다.

빛·에너지 생명체(31~40구역 주력, "빛 계열" - 곡선·발광·반투명으로 각진 기계와도 확 다르다, 상세 docs/12):
- **wisp(도깨비불)**: 지그재그(사인)로 부유하며 내려오고, 일정 주기마다 작은 자식 도깨비불을 낳는다(최대 2). 약하고 빠르며, 자식은 재분열하지 않는다.
- **jelly(빛해파리)**: 느리게 하강하며 늘어진 촉수를 좌우로 흔든다. 발사하지 않고 접촉 피해만 - 대신 크고 단단하다.
- **bloom(빛꽃)**: 잠깐 내려오다 멈춰 사방으로 방사형 탄을 '개화'하고, 다시 내려오기를 반복한다. 꽃잎(탄) 수는 난이도(`radialMul`)에 따라 준다.
- **whale(빛고래)**: 크고 느린 유영체. 좌우로 곡선 유영하며 내려오고 가끔 조준 3연발을 쏜다. 발광하는 약점 코어. 매우 단단하다.

보너스 기체:
- **bonus**: 주기적으로 화면 좌/우에서 등장해 가로질러 간다. 파워업의 주 공급원 - 격파 시 확정 드롭(잡몹은 드롭 없음).

- 적 체력은 구역이 오를수록 상승: `base × (1 + (stage-1)×enemyHpScale)`, 11구역부터 추가 배수(`hardStage.hpMul`), 21구역부터 `voidStage.hpMul`, 31구역부터 `aeonStage.hpMul`이 차례로 곱해진다. 3계통 화력 성장과 균형.
- 상세 수치 → 02_data.md.

## 6. 보스 (부위 파괴형 - 상세 06_boss.md)

- 모든 보스는 **코어(본체) + 부위** 구조다. 정면 화력(메인·사이드)은 살아있는 부위를 먼저 때리고, 방어구가 남아있으면 코어는 피해를 받지 않는다(체력 바 "보호 중"). **유도탄·에너지존은 부위를 무시하고 코어를 직접** 때린다(shielder 공략과 일관).
- 부위 종류: **포탑(weapon)** = 자기 공격 패턴 보유(부수면 그 패턴 영구 정지 - 탄막이 줄어드는 보상), **방어구(shield)** = 코어를 가림(다 부수면 코어 노출). 부위를 부술 때마다 점수.
- 스타일 4종(구역대별 다른 모습·공략):
  - **1~10 함선형**: 좌우 포탑 2개, 방어구 없어 코어 처음부터 노출. 포탑을 부수면 탄막이 준다(입문).
  - **11~20 생체형**: 촉수 방어구 3개가 코어를 가린다. 촉수를 다 부숴야 코어 노출.
  - **21~30 위성형**: 코어 주위를 도는 실드 조각 4개. 조각을 떼어낼수록 코어 노출 시간이 는다(타이밍 사격).
  - **31~39 순환 재활용**: 빛 생명체 구간의 중보스는 위 세 부위파괴형(함선→생체→위성)을 순서대로 재활용한다(`bossStyleFor`). 새 스타일은 만들지 않는다.
  - **40 파수꾼형(최종)**: 머리·양 어깨 포탑 + 가슴 방어판. 부위를 부술수록 남은 포탑이 격화(광폭화).
- **강화판(각 10구역 뒤 5개, 2026-07-16)**: 6~10·16~20·26~30·36~39구역은 앞 5개(1~5·11~15·21~25·31~35)의 강화판이다(`bossUpgradeFrom`, `(stage-1)%10 >= 5`). 부위가 하나 더 붙고 발사 주기가 짧아진다. 총 체력은 (코어+전체 부위) 비율 합으로 정규화해 일정하게 유지한다. 등장 배너에 "(강화)" 표시. 40(최종)은 강화 대상이 아니다.
- 좌우 유영 + 호위 비행체 주기 소환(중보스). 격파 시 파워업: 중보스 2개, 최종보스 4개.
- 등장 위치는 상단 체력 바 아래(`boss.spawnTop`)로 내려 바와 겹치지 않는다.

## 7. 파워업 아이템 (6종)

- **P**: 전방 화력 +1 (1~6 탄 수, 7~66 = 가운데 탄부터 발별 순차 진화, 각 탄 10단계). 최대(66) 후엔 점수 보너스.
- **S**: 옵션기 1대 추가 (최대 6) → 그 뒤 안쪽 총알부터 발별 순차 진화(각 발 10단계).
- **E**: 에너지존 레벨 +1 (최대 5).
- **T**: 꼬리 비행기 성장 - 대수 +1(4대까지) → 그 뒤 앞 비행기부터 발별 순차 무기 진화(각 대 10단계). 최대 후엔 점수 보너스.
- **H**: 목숨 +1(난이도별 최대). 친구 동행 중이면 친구 회복·부활도 공유(09_friend.md).
- **B**: 봄(폭탄) - 화면의 적 전멸 + 보스 코어에 큰 데미지. 아이템은 도화선·불꽃 달린 폭탄 모양으로 그려 폭탄임을 알아보게 하고, 먹으면 화면에 은은한 폭발 섬광이 번진다(`bombFlash`). 봄으로 죽은 **보너스 기체는 파워업을 확정 드롭**한다(잡몹은 드롭 없음 - 봄이 과해지지 않게).
- 보스·보너스 기체는 확정 드롭, 일반 잡몹은 낮은 확률(`drop.chance`)로 드롭한다(초반 화력 성장 숨통).
- 피격 시: 목숨 -1 + 마지막 얻은 파츠(P/S/E/T, 진화 단계 포함) 1개 손실(역순).
