# 08. 21~30 스테이지 확장 (심층 우주 - 이질 적)

사용자 지시(2026-07-09): "30 스테이지까지 추가. 완전 다른 형태의 적들이어야 해." 이 문서는 20→30 스테이지 확장 + 21~30 전용 신규 적의 **설계안**이다. 아직 설계 단계이며(구현은 확정 후), 이 문서가 구현의 SSOT가 된다.

## 0. 요약

- 스테이지 20 → **30**. 21~30은 지금까지 없던 **완전 다른 형태의 적**으로 채운다.
- 기존 적 8종은 전부 **둥근 유기체·정령류**(도깨비불·나비·먼지·방패 정령 등)다. 21~30 신규 적은 그 반대편 - **각지고 대칭적인 기계·결정·에너지 계열**로 만들어 시각·행동 모두 확 다르게 한다.
- 최종보스를 20구역 → **30구역**으로 이동. 21~29는 중보스 구간이 된다.

## 1. 현황

- `stageCount` 20, `STAGE_NAMES` 20개(1 소행성 지대 ~ 20 최종 결전).
- 적 8종: 1~10 기본(drone 도깨비불 / weaver 나비 / gunner 먼지) + 11~20 신규(splitter 분열 / shielder 방패 / rusher 돌격) + bonus 보너스 기체.
- 웨이브: `buildWaves`(1~10) / `buildHardWaves`(11~20, `hardStage.from`=11). 최종보스는 `spawnBoss`가 `stage >= stageCount`로 판정.
- 전부 둥근 실루엣 + 눈(정령 톤 통일). "완전 다른 형태"는 이 톤을 깨는 신규 계열이 필요.

## 2. 목표

- `stageCount` 30. 21~30 전용 웨이브 빌더(`buildVoidWaves`) + 신규 적 계열.
- "완전 다른 형태"의 정의: (1) 실루엣이 각지거나 기하학적(삼각·육각·체인) (2) 눈이 없거나 기계적 코어 (3) 행동이 기존과 다른 축(정지 포화·반사·연결선·자폭·순간이동 등).

## 3. 신규 적 (제안 6종 — 채택 개수·컨셉 사용자 확정 필요)

각 적은 **형태 / 행동 / 대략 hp·속도 / 역할**로 정의한다. 색은 기존(따뜻한 빨강·주황 계열 적)과 구분되게 **차가운 기계색(강철 청·전기 하늘·형광 자주)**을 쓰되 아군 시안과도 안 겹치게 조정.

### 3.1. turret (부유 포대)
- **형태**: 육각 기계 몸체 + 회전 포신. 눈 없음, 중앙 코어 발광.
- **행동**: 상단 근처에서 거의 정지(아주 느린 하강), 포신이 플레이어를 조준해 3방향 연사. "치우지 않으면 계속 쏘는 고정 화력".
- **hp·속도**: hp 높음(단단), 속도 매우 느림. 점수 높음.

### 3.2. prism (결정체)
- **형태**: 삼각/마름모 결정(투명감). 
- **행동**: 직하하되 **피격 시 사방으로 반사탄** 1~2발 튕김(함부로 못 쏘게). 
- **hp·속도**: hp 중간, 속도 중간.

### 3.3. coil (전격 코일)
- **형태**: 노드 2개가 **전기 아크 라인**으로 연결된 쌍.
- **행동**: 쌍이 나란히 내려오며, 두 노드 사이 아크 선에 플레이어가 닿으면 피해(선 자체가 위험). 노드 하나 부수면 아크 사라짐.
- **hp·속도**: 노드별 hp 낮음, 속도 중간. 위치 잡기 압박.

### 3.4. serpent (기계 뱀)
- **형태**: 여러 마디가 이어진 체인. **머리만 약점**(몸통 마디는 무적/방어).
- **행동**: 구불구불(사인파) 하강, 머리를 노려야 격파. 몸통이 길어 회피 압박.
- **hp·속도**: 머리 hp 중상, 속도 중간. (구현 복잡도 상 - 미채택 시 후순위)

### 3.5. mine (부유 기뢰)
- **형태**: 팔각 기뢰 + 돌기.
- **행동**: 느리게 표류하다 플레이어가 근접하면 **자폭(범위 파편탄)**. 정면 돌파를 벌한다.
- **hp·속도**: hp 낮음, 속도 매우 느림.

### 3.6. warper (공간 왜곡체)
- **형태**: 불안정한 마름모 + 잔상.
- **행동**: 짧게 **순간이동**하며 하강(위치 예측 어려움). 이동 직후 짧게 취약.
- **hp·속도**: hp 중간, 순간이동 사이 정지.

> 권장: 구현·검증 부담을 고려해 **4종 채택**(turret·prism·mine·warper)을 1순위로 제안한다. coil(연결선 판정)·serpent(체인)는 구현 난도가 높아 2차로 미루거나 범위를 줄이는 걸 권장.

## 4. 스테이지 21~30

### 4.1. 이름 재배치
현재 20 "최종 결전"이 마지막인데, 30 확장 시 20은 중간이 된다. **20의 이름을 바꾸고 "최종 결전"류를 30으로** 옮긴다. 21~30 제안(심우주 이질 문명 톤):

| 구역 | 이름(제안) |
|---|---|
| 20 | 심연의 관문 (기존 '최종 결전'에서 변경) |
| 21 | 기계 성역 |
| 22 | 결정 동굴 |
| 23 | 방전 회로 |
| 24 | 기뢰 지대 |
| 25 | 왜곡 성운 |
| 26 | 강철 함대 |
| 27 | 침묵의 기계신 |
| 28 | 무한 회랑 |
| 29 | 종말 관측소 |
| 30 | 최종 결전 |

### 4.2. 웨이브·보스
- `buildVoidWaves(stage)` 신설: 21~25 신규 적 도입(구역당 1~2종씩 늘림), 26~30 신규+기존 혼합 고밀도.
- **보스**: 21~29 중보스, 30 최종보스. 중보스 외형은 06_boss.md(부위 파괴형) 스타일을 따르는 게 이상적이나, 06 미구현 상태면 이번엔 기존 mini 보스 재사용 + 색·hp만 강화(설계 결정 필요).

## 5. 밸런스

- **hp 스케일**: 21~30은 4계통이 거의 만렙에 근접하는 구간이라 DPS가 최대다. `hardStage`(11~) 배수 위에 **21~30 추가 배수**(예: `voidStage.from`=21, `hpMul` 1.6~1.8)를 얹어 순삭을 막는다. dev 훅으로 실측 후 확정.
- **신규 적 수치(초안, 조정 대상)**:

| 적 | hp(base) | 속도 | 점수 | 특징 |
|---|---|---|---|---|
| turret | 6 | 40 | 400 | 정지·3방향 연사 |
| prism | 4 | 110 | 350 | 피격 시 반사탄 |
| mine | 2 | 45 | 300 | 근접 자폭 범위탄 |
| warper | 3 | 0(순간이동) | 380 | 순간이동 하강 |
| coil(노드) | 2 | 100 | 250 | 아크 선 피해 |
| serpent(머리) | 5 | 90 | 450 | 머리만 약점 |

- 신규 적도 구역 스케일(`scaledHp`) + void 배수 적용.

## 6. 바꿀 파일 (구현 단계)

- `data/numbers.js` - `stageCount` 30, `STAGE_NAMES` 30개(20 이름 변경), `enemy`에 신규 적 수치, `voidStage`(from 21·hpMul) 신설, 신규 적별 파라미터(turret shots/spread, prism 반사 수, mine 폭발 반경, warper 순간이동 주기).
- `data/colors.js` - 신규 적 6색(기계 청·전기 하늘·형광 자주).
- `core/waves.js` - `buildVoidWaves(21~30)` + `buildWaves`에서 `stage >= voidStage.from`이면 분기.
- `core/spawn.js` - `spawnEnemy` 신규 적 초기화, coil 쌍/serpent 마디 특수 스폰, 최종보스 `stage >= 30` 판정 유지.
- `core/world.js` - `updateEnemies`에 신규 적 행동(turret 조준 연사·prism 반사·mine 자폭·warper 순간이동·coil 아크 판정), 자폭/반사탄은 `enemyFireAt`류 재사용.
- `render/view.js` - `drawEnemies`에 신규 형태 6종(각진·기계·결정).
- `tests/` - 신규 적 스폰·행동 단위, `buildWaves(21~30)` 유효, 최종보스 30구역, void hp 스케일.
- `docs/01_spec.md` 5장(적)·`02_data.md` 갱신.

**확정(사용자 지시 2026-07-09)**:
- 30 스테이지로 확장, 21~30은 완전 다른 형태의 적.
- **신규 적 4종 채택**: turret(부유 포대) / prism(결정체) / mine(기뢰) / warper(공간 왜곡체). **coil·serpent는 2차**(연결선·체인 구현 난도 높아 이번 범위 제외).
