# 09. 친구 비행기 (어린이 모드 전용)

어린이 모드에서만 등장하는 동료 비행기 설계. 사용자 지시(2026-07-14) 기반. SSOT - 코드보다 이 문서가 진실.

## 1. 개요

1.1. 어린이 모드로 게임을 시작하면 화면 옆에서 친구 비행기가 말풍선으로 인사하며 날아 들어와 함께 싸운다.
1.2. 일반 모드에는 등장하지 않는다(`game.friend` = null).
1.3. 목적: 어린이가 혼자가 아니라 동료와 함께 논다는 안심감 + 옆에서 구경하는 재미(내 것과 완전히 다른 총알이 넓게 퍼지는 걸 본다).

## 2. 등장 연출

2.1. 어린이 모드 시작 시 화면 왼쪽 밖에서 플레이어 옆으로 날아 들어온다.
2.2. 들어오는 동안(`enterTime`) 말풍선을 순서대로 띄운다: "안녕!" → "난 친구야" → "같이 게임하자!"(`CFG.friend.speech`, 각 `speechEach`초).
2.3. 말풍선은 캔버스에 친구 위쪽 둥근 사각형으로 그린다(HUD 아님). 3개 다 지나가면 사라진다.
2.4. 등장(enter) 중에는 발사하지 않는다. 자리 잡은 뒤 자동발사 시작.

## 3. 이동 (자율 유영)

3.1. 플레이어 옆(`sideOffset` 왼쪽)에 떨어져 따라온다. 살짝 위(`upOffset`) + 위아래 흔들림(`bob`).
3.2. 가까운 적 쪽으로 좌우로 스스로 끌린다(`aimBias`). 단 플레이어에서 `maxDist` 이상 벗어나지 않고 화면 안에 머문다.
3.3. 조작 불가(플레이어만 조작). 친구는 항상 AI 자율.

## 4. 무기 (메인 총알만)

4.1. 친구는 사이드 총알·꼬리 비행기·에너지존이 없다. **메인 총알만** 있다.
4.2. 메인 총알은 강화 10단계(`level` 0~10). 단계가 오를수록 **부채꼴로 더 넓게, 더 많이** 쏜다 - 플레이어(직진 나란히)보다 넓게 퍼진다.
   - 발 수 = `shotsBase + level*shotsPerLevel`(1~11발), 총 부채각 = `spreadBase + level*spreadPer`(도).
   - 발당 데미지 = `dmgBase + level*dmgGrow`, 탄 크기 = `bulletR + level*bulletRGrow`.
4.3. 총알 모양은 플레이어(차가운 색 레이저 빔)와 **완전히 다르다**: 따뜻한 색(골드·핑크 계열) 별/반짝이 모양(`drawFriendShot`). 적탄(작은 빨간 원)과도 색·형태·크기로 뚜렷이 구분.
4.4. 친구의 강화 정보는 HUD에 따로 표시하지 않는다(옆에서 넓어지는 걸 눈으로 볼 뿐).

## 5. 아이템 공유 (서로서로 강화)

5.1. 친구와 플레이어는 아이템을 공유한다. 친구가 먹어도 플레이어가 강화되고, 플레이어가 먹어도 친구가 강화된다.
5.2. 아무 강화 아이템(P·S·E·T)이든 누가 먹든:
   - 플레이어는 그 아이템 계통이 오른다(기존 gainFront/Option/Zone/Tail 그대로).
   - 친구 메인 총알 `level`도 +1(최대 10). = 모든 아이템 공유.
5.3. 친구가 만렙(10)이어도 플레이어 계통이 오르면 정상 강화. 둘 다 무의미할 때만 점수 보너스(`maxedBonus`).
5.4. 아이템 획득 판정: 플레이어 또는 친구가 아이템에 닿으면 1회 획득(양쪽 동시 닿아도 한 번).

## 6. HP / 피격

6.1. 친구는 자기 하트 5개(`maxHp`). 플레이어 목숨과 별개 풀.
6.2. **총알·충돌 피해는 각자 개별**: 친구가 맞으면 친구 hp만 -1(플레이어 목숨 불변), 플레이어가 맞으면 플레이어만(기존 그대로). 피격 후 짧은 무적(`inv`).
6.3. 친구 hp 0 → 기절(`down`, 사라짐). 게임오버 아님 - 플레이어 혼자 계속.
6.4. **회복(H)은 공유**: H를 먹으면 플레이어 목숨 +1(상한 내) + 친구 hp +1(상한 내, 살아있을 때).
6.5. 친구가 기절한 상태에서 H를 먹으면 **부활**(hp 풀 회복 + 옆으로 재등장, `reviveMsg` 말풍선). 강화 `level`은 유지(모아둔 성장은 사라지지 않음).

## 7. 점수

7.1. 점수는 원래 하나(`game.score`)라 자동 공유. 친구 총알로 잡은 적도 같은 점수에 합산.

## 8. 구현 위치 (매직넘버 0 / core 순수)

8.1. 수치: `src/data/numbers.js` `CFG.friend`. 색: `src/data/colors.js` `COLORS.friend`.
8.2. 순수 로직: `src/core/friend.js` - `spawnFriend`/`stepFriend`(이동·말풍선·발사)/`gainFriendLevel`/`friendTakeHit`/`reviveFriend`/`nearestEnemyX`. DOM/Canvas 미의존, `game.sfx`로만 사운드 신호.
8.3. 월드 연동: `src/core/world.js` - `stepWorld`에서 `stepFriend` 호출, `checkCollisions`에서 친구 피격·아이템 공유(`grabItem`) 처리.
8.4. 렌더: `src/render/view.js` - `drawFriend`(몸·hp 점·말풍선)/`drawFriendShot`(별 총알)/`drawBullets`에 `kind:'fmain'` 분기.
8.5. 생성: `src/main.js` - 어린이 모드 startGame 시 `spawnFriend`. 일반 모드는 null 유지.
8.6. 총알 `kind: 'fmain'`. 데미지·충돌은 기존 정면 화력(메인/사이드)과 동일 경로(부위·코어 타격, 방패병에 막힘) - 별도 특례 없음.
