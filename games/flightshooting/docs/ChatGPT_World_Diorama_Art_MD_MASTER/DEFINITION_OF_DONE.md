# DEFINITION OF DONE

## 완료 원칙

다음 조건을 모두 만족하고 사용자가 명시적으로 승인한 경우에만 국가 배경 제작을 완료로 본다.

AI의 자체 판단만으로 완료 처리하지 않는다.

검토 중인 시안, 임시 이미지, 자동 생성 결과, 기술적으로 출력이 끝난 이미지는 사용자 승인 전까지 완료본이 아니다.

---

## 시각 기준

- 최신 승인 기준 이미지와 같은 카메라 느낌과 공간감을 유지한다.
- 국가 전체가 하나의 연속된 미니어처 디오라마로 보인다.
- 전통, 현대, 자연, 주거, 산업 지역이 콜라주처럼 끊기지 않는다.
- 화면 아래에서 위로 하나의 세계가 자연스럽게 이어진다.
- 주요 랜드마크를 실루엣과 구조만으로 식별할 수 있다.
- 랜드마크가 관광 포스터처럼 과도하게 크거나 균등하게 나열되지 않는다.
- 국가별 렌더 스타일과 카메라가 프로젝트 기준에서 벗어나지 않는다.

---

## 플레이 기준

- 중앙에 연속된 플레이 공간이 확보되어 있다.
- 중앙 이동축이 비어 보이지 않으면서도 전투 가독성을 유지한다.
- 배경이 플레이어, 적, 탄환보다 강하게 보이지 않는다.
- 중앙에 전투를 방해하는 대형 구조물이나 강한 경계가 없다.
- 밝은 반사와 점광원이 탄환처럼 보이지 않는다.
- 실제 플레이 배율에서 플레이어, 적, 탄환을 즉시 구분할 수 있다.
- 세로 스크롤 중 랜드마크와 배경이 전투 공간을 가리지 않는다.

---

## 인트로 기준

- 인트로와 플레이가 동일한 공간을 사용한다.
- 인트로 랜드마크가 줌아웃 후에도 같은 배경 안에 자연스럽게 남는다.
- 별도 인트로 이미지나 독립된 시네마틱 장면을 만들지 않는다.
- 인트로 확대 화면과 플레이 축소 화면의 구조가 일치한다.
- 인트로용으로 랜드마크 위치나 카메라를 따로 재배치하지 않는다.

---

## 교육 기준

- 플레이 중 핵심 랜드마크를 인지할 수 있다.
- 국가를 대표하는 지리적 특징이 환경 안에 드러난다.
- 국가를 대표하는 문화적 특징이 환경 안에 드러난다.
- 설명 문구 없이도 최소 하나 이상의 기억 포인트가 남는다.
- 교육 요소가 전투 가독성을 해치지 않는다.
- 랜드마크가 단순 장식이 아니라 국가를 구분하는 시각적 단서로 작동한다.

---

## 스크롤 기준

- 화면이 아래에서 위로 자연스럽게 이어진다.
- 상단과 하단이 포스터의 시작과 끝처럼 닫히지 않는다.
- 청크 분할 후에도 지형, 강, 도로, 해안선, 건물, 조명이 끊기지 않는다.
- 핵심 랜드마크가 청크 경계에 걸리지 않는다.
- 청크별로 카메라, 색감, 시간대가 달라지지 않는다.
- 실제 스크롤에서 장면이 독립된 패널처럼 보이지 않는다.

---

## 출력 기준

- UI, 텍스트, 라벨, 아이콘, 버튼이 없다.
- 전투기, 적, 탄환, 폭발, 데미지 숫자가 없다.
- 정확히 하나의 환경 배경 이미지만 출력한다.
- 승인된 원본과 수정본이 구분되어 있다.
- concept, production, review, chunk 파일이 구분되어 있다.
- 승인 후 요청되지 않은 추가 재해석이나 수정이 없다.
- 최신 승인 기준 이미지와 사용한 버전을 확인할 수 있다.

---

## 승인

- 사용자가 명시적으로 승인한 결과만 완료로 인정한다.
- 수정 요청이 남아 있으면 완료 상태로 기록하지 않는다.
- AI가 품질이 충분하다고 판단해도 사용자 승인 없이 완료 처리하지 않는다.
- 승인된 결과는 production 승인본으로 기록한다.
- 승인일과 버전을 CHANGELOG.md에 남긴다.

---

## 실패 조건

다음 중 하나라도 발생하면 완료가 아니다.

- 최신 승인 기준 이미지와 다른 카메라
- 숫자 각도를 근거로 기준 카메라를 재해석한 결과
- 독립 장면을 이어 붙인 콜라주 형태
- 관광 포스터형 구성
- 완전한 탑다운, 측면, 평면 지도 또는 아이소메트릭 타일맵
- 중앙 플레이 공간 부족
- 중앙을 막는 대형 랜드마크 또는 구조물
- 플레이어, 적, 탄환보다 강한 배경
- 랜드마크 식별 불가
- 인트로와 플레이 공간 분리
- 별도의 인트로 배경 사용
- 스크롤 또는 청크 경계 단절
- 지역별 카메라, 시간대, 렌더 스타일 불일치
- UI, 텍스트, 라벨 포함
- UI, 전투기, 적, 탄환, 폭발 등 게임플레이 오브젝트 포함
- 승인되지 않은 이미지의 기준 이미지 사용
- 사용자 미승인
