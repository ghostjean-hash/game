# Blessed Lotto

캐릭터 시드 + 12종 추첨 전략 기반 한국 6/45 로또 번호 추천 PWA.
**당첨 확률 향상이 아니라 선택 행위의 서사화**가 목적.

## 1. 플레이

- 게임 허브 루트의 Blessed Lotto 카드 클릭.
- 또는 직접 `games/lotto/`.
- 첫 진입: 면책 안내 → 캐릭터 생성 폼 → 메인.

## 2. 조작

- 터치 / 클릭으로 슬롯 / 전략 / 회차 / 통계 / 전적 / 휠링 이동.
- 키보드: Tab 이동, Enter 확인, ESC 모달 닫기.

## 3. 메커닉

### 3.1. 캐릭터 (정체성)
- 이름 / 생년월일 / 별자리 / 행운 단어로 시드 생성 (FNV-1a).
- 자동 추출: 12간지(생년) / 일주(사주, 생년월일) / 별자리 4원소.
- MBTI는 선택 입력.
- Luck 0~100 (등수 적중 시 1회 보너스).

### 3.2. 추첨 전략 12종 (사용자가 매번 선택)

4카테고리로 분류 (콘텐츠 차원, docs/02_data.md 1.5.2):

| 카테고리 | 전략 | 가중치 |
|---|---|---|
| **통계** | 통계 추첨 | 본번호 누적 빈도 (power 보정) |
| **통계** | 보너스볼 사냥 | 본번호 + 보너스 모두 보너스볼 빈도 |
| **통계** | 미출현 회귀 | 미출현 갭 (power 보정) |
| **통계** | 짝꿍 번호 | 시드 키번호 동시출현 페어 |
| **통계** | 최근 트렌드 | 최근 30회 Hot (power 보정) |
| **운세 매핑(서양)** | 별자리 행운 | 12별자리 행운 번호 |
| **운세 매핑(서양)** | MBTI 행운 | 16종 행운 번호 |
| **운세 매핑(서양)** | 별자리 4원소 | 불/땅/공기/물 그룹 행운 번호 |
| **사주(동양)** | 일주 오행 | 캐릭터 일주 천간 오행(목/화/토/금/수) 행운 번호 |
| **랜덤** | 축복받은 자 | 시드 + Luck 분산 |
| **랜덤** | 직감 | 매 회차 다른 무작위 (결정론) |
| **랜덤** | 균형 조합 | 합 121-160 / 홀짝 3:3 필터 통과 |

결정론 차원: 객관 5(캐릭터 무관) / 시드 의존 7(캐릭터별 다른 결과). 자세한 내용은 `docs/02_data.md` 1.5.1.

### 3.3. 운세 (4등급)
- 1단계: 캐릭터 띠 + 회차 일진 → 4가지 관계별 분포.
- 2단계: 캐릭터 일주 + 회차 일주 → 오행 6관계 ±보정.
- 흉일에는 카드 외곽 빨강 + "방어 모드 권장" 배너.

### 3.4. 휠링 (다구좌 모드, 옵션)
- 메인 하단 토글 ON 시 활성. 첫 ON 시 윤리 안내 모달 강제.
- Full Wheel: 풀의 모든 6 조합. 4-if-4 자동 보장.
- Abbreviated Wheel: 시드 기반 N장. 4-if-4 보장 자동 검증.

## 4. 로컬 실행

### 권장: 정적 dev 서버
저장소 루트에서:

```
node scripts/dev-server.mjs 8000
```

브라우저: `http://127.0.0.1:8000/games/lotto/`

### Live Server (VS Code)
포트 5500. 동작하지만 캐시 / SW 충돌 가능. dev-server 권장.

## 5. 회차 데이터 페치 (처음 한 번)

1회차부터 최신 회차까지 smok95/lotto 미러(MIT, 매주 토 자동 갱신)에서 받아옵니다. all.json bundle 단일 GET, **1초 미만**.

### Windows 더블클릭
`scripts/fetch-lotto-draws.bat` 더블클릭.

### 또는 명령어
```
node scripts/fetch-lotto-draws.mjs
```

### 자동 갱신
GitHub Actions가 매주 일요일 03:00 KST에 자동 페치 + push.

### 페치 없이 사용 가능한 전략
축복받은 자 / 짝꿍 번호 / 별자리 행운 / 직감 / 균형 조합 / MBTI 행운 / 별자리 4원소 / 일주 오행.

- 빌드 시스템 없음 (정적 파일).

## 8. 윤리

- "확률 향상" 표현 금지. 추천은 **참고용**.
- 휠링은 다구좌 모드 ON 시에만 노출 + 첫 활성화 윤리 안내 모달.
- 결제 가능 항목: 캐릭터 슬롯 / 스킨 (현재 미도입).
- 도박 권유 아님.

## 9. 적용 표준

- `html-game v0.2` (`standards/html-game/`).
- 매직 넘버 0개 (모든 수치는 `src/data/numbers.js`).
- 게임 색상 = `src/data/colors.js`. UI 토큰 = `styles/tokens.css`. 혼용 금지.

자세한 내용은 `docs/01_spec.md` ~ `docs/04_conventions.md` 참조.
