# Lotto 5세트 묶음 편향 검증 보고서 (N1 가설)

- **생성일**: 2026-05-10
- **표본**: 60명 캐릭터 × 200묶음 = 캐릭터당 200묶음(=1000세트), 전략당 12000묶음
- **묶음 정의**: 사용자 "+5세트" 버튼 1회 = 5세트 × 6번호 = 30번호
- **회차 mix**: 최근 100회 (drwNo 1124 ~ 1223)
- **자연 baseline**: 균등 6/45 추첨으로 5000묶음 시뮬한 결과 (재현성 고정 시드)
- **판정 기준 (K3)**: K1(자연 평균 ±20%) + K2(z-test p<0.05). 둘 다 위반 = **강한 편향**, 한쪽 = **주의**, 둘 다 통과 = **정상**.

## 4. 자동 결론

자비스 자동 분류 = **결손 의심 없음**. 5세트 묶음 단위에서도 모든 측정 차원이 자연 baseline 범위 내.

> [의견] 본 보고서는 1차(개별 본번호 분포) → 2차(5세트 묶음 인상) 진단의 두 번째. 두 보고서 모두 결손 없음이면 사용자 인상의 출처는 시각 인지(번호공 색)나 N=1~5 표본의 통계적 변동일 가능성이 높음.
