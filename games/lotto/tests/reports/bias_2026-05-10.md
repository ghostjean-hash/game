# Lotto 추천 번호 편향 검증 보고서

- **생성일**: 2026-05-10
- **표본**: 60명 캐릭터 × 1000회 추출 = 캐릭터당 1000세트, 전략당 60000세트
- **회차 mix**: 최근 100회 (drwNo 1124 ~ 1223)
- **판정 기준 (K3)**: K1(자연 기대 ±20%) + K2(카이제곱 p<0.05). 둘 다 위반 = **강한 편향**, 한쪽 = **주의**, 둘 다 통과 = **정상**.
- **자연 기대 (6/45 균등)**: 합 평균 138.0 / 합 표준편차 29.95 / 인접 페어 52.87% / 자카드 0.0755
- **kind 'random'** = 자연 기대 균등에 가까워야 정도(正道). **kind 'biased'** = 학설/통계 풀에 의도된 편향.
