# Rush Hour

6x6 주차장에서 앞뒤 차를 밀어, 가로막힌 빨간 차를 오른쪽 출구로 빼내는 슬라이딩 블록 퍼즐.

## 플레이

- 게임 허브에서 Rush Hour 카드를 누르면 진입.
- 차를 잡고 그 차의 방향(가로차는 좌우, 세로차는 상하)으로 끌어 민다.
- 길을 터서 빨간 차를 오른쪽 출구로 빼내면 클리어.
- 적은 이동수로 풀수록 좋다. 퍼즐마다 최소 이동수가 목표로 표시된다.

## 조작

- 터치 / 마우스 드래그. 차를 끌면 막히는 곳까지 미끄러지고, 손을 떼면 칸에 맞춰 멈춘다.
- 되돌리기 / 리셋 / 이전·다음 퍼즐 버튼.

## 로컬 실행

게임 허브 루트에서 정적 서버를 띄운다.

```
node scripts/dev-server.mjs 8000
```

브라우저에서 `http://127.0.0.1:8000/games/rushhour/`.

## 테스트

`tests/test.html`을 브라우저로 열면 전체 테스트가 자동 실행되고 상단에 PASS / FAIL이 표시된다. 모든 내장 퍼즐이 풀 수 있는지(솔버 검증)까지 자동 확인한다.
