# 01 사양 (Rush Hour)

## 0. 컨셉 / 테마

0.1. 대상은 초등 저학년. 쉽고 귀엽고, 성취감을 자주 느끼게 하는 것이 최우선이다.
0.2. 비주얼은 파스텔 톤의 둥글고 귀여운 오리지널 동물 캐릭터(산리오풍 느낌). 실제 산리오 등 상표 캐릭터 이미지는 쓰지 않는다.
0.3. 차 = 동물 친구. 블록 크기로 동물이 정해진다: 가로 2칸 친구=고양이, 가로 3칸=강아지, 세로 2칸=병아리, 세로 3칸=펭귄. 주인공(target, 기존 빨간 차)은 토끼 고정. 같은 종류라도 표정(방긋 / 윙크 / 동그란 눈 / 놀람)·색·작은 액세서리(리본 / 나비넥타이 / 꽃)가 블록 위치마다 달라 다양하게 보인다.
0.4. 보드는 파스텔 마을 길, 출구는 오른쪽 밖으로 이어지는 밝은 출구 길과 게이트(별·화살표 표시)다.
0.5. 스토리: 친구들이 길을 막고 있다. 친구들을 살살 비켜 주며 주인공 토끼를 출구 밖으로 내보내면 클리어. 토끼가 길을 따라 빠져나가고 별·하트 축하 연출이 나온다.
0.6. 난이도는 쉬움 위주(최소 수 1~10). 13수 이상의 어려운 퍼즐은 넣지 않는다.

## 1. 한 줄

6x6 마을 길에서 동물 친구들을 밀어 길을 터, 주인공 토끼를 집(오른쪽 출구)까지 데려다주는 슬라이딩 블록 퍼즐.

## 2. 코어 루프

1. 퍼즐 하나가 보드에 차들로 배치된다.
2. 플레이어가 차를 드래그해 그 차의 방향(가로차는 좌우, 세로차는 상하)으로 민다.
3. 차는 빈 칸이 있는 만큼만 미끄러지고, 다른 차나 벽에 막히면 멈춘다.
4. 길을 터서 빨간 차를 오른쪽 출구까지 빼내면 클리어.
5. 다음 퍼즐로 진행. 더 적은 수로 풀수록 좋다.

## 3. 규칙

3.1. 보드는 6x6 격자. 좌표는 (row, col), 둘 다 0부터 5.
3.2. 차는 길이 2(승용차) 또는 3(트럭), 방향은 가로 또는 세로.
3.3. 빨간 차(target)는 항상 가로, 길이 2, 출구 행(row=2)에 놓인다.
3.4. 출구는 보드 오른쪽 가장자리, 출구 행(row=2)의 우측.
3.5. 차는 자기 방향으로만 이동한다. 가로차는 같은 행에서 좌우, 세로차는 같은 열에서 상하.
3.6. 이동 경로에 다른 차가 있으면 그 앞까지만, 벽이면 가장자리까지만 미끄러진다.
3.7. 클리어 = 빨간 차의 오른쪽 끝이 보드 오른쪽 가장자리에 닿는 것(col + length == 6).

## 4. 조작

4.1. 주 조작은 드래그(터치 / 마우스 공통). 차를 잡고 이동 방향으로 끌면 그 방향으로 미끄러진다.
4.2. 드래그 중에는 이동 가능 범위 안에서만 따라오고, 손을 떼면 손을 뗀 자리에서 가장 가까운 칸으로 점프 없이 부드럽게 정착한다.
4.3. 이동 방향과 직각인 드래그는 무시한다(가로차를 위아래로 끌어도 움직이지 않음).
4.4. 끌지 않고 짧게 탭하면 누른 지점이 차의 어느 쪽인지(이동 축 기준 절반)에 따라 그 방향으로 한 칸 민다. 막혀 있으면 움직이지 않는다(이동 0은 수로 치지 않는다).

## 5. 이동수 카운트

5.1. 차 한 대를 한 방향으로 한 번 미는 것이 1수다(몇 칸을 가든 1수). 러시아워 표준 카운팅.
5.2. 같은 차를 멈췄다가 다시 밀면 별개의 수다.
5.3. 각 퍼즐의 최소 이동수는 솔버(BFS 최단 탐색)가 계산해 목표로 표시한다.

## 6. 성공 / 실패 / 종료

6.1. 성공 이벤트: 토끼(빨간 차) 출구 도달 = 퍼즐 클리어. 토끼가 출구 길로 미끄러져 나가는 애니메이션 + 별·하트 파티클이 터진 뒤 결과 오버레이가 뜬다.
6.2. 실패 이벤트: 없음(이동수 제한 없음, 막히면 되돌리면 됨).
6.3. 종료 조건: 없음(퍼즐 컬렉션을 모두 클리어하면 완주, 계속 재도전 가능).
6.4. 제한시간: 퍼즐별 = 최소 수 × 20초 + 60초(넉넉). 시간이 지나도 게임 오버 없이 계속 풀 수 있다(실패 없음 유지). 시간 안에 클리어하면 보너스 골드.
6.5. 보상: 별(최소 수 이내 ⭐⭐⭐ / +3수 이내 ⭐⭐ / 그 외 ⭐) + 골드(기본 10 + 별 보너스 3별 20·2별 10 + 시간 내 클리어 10 + 연속 콤보 보너스 §6.7). 골드는 누적 저장하고, 상점(토끼 색 스킨)과 힌트(§7.6)에서 쓴다. 퍼즐별 최고 별 기록.
6.6. 주인공 토끼 표정: 탈출 전 무표정 → 제한시간 50% 경과 시 어두운 표정 → 85% 경과 시 울상. 탈출(클리어) 시 활짝 웃음.
6.7. 연속 콤보: 제한시간 안에 연달아 클리어하면 콤보가 1씩 쌓인다. 2연속부터 콤보 보너스 골드를 추가로 준다(콤보가 길수록 커지고 상한이 있다). 시간을 초과해 클리어하면 콤보가 끊겨 0이 된다(클리어 자체는 인정, 실패 없음 유지). 클리어 결과 화면에 "🔥 N연속"과 보너스를 보여준다. 콤보와 최고 콤보는 저장한다(02_data §4).

## 7. 진행 시스템

7.1. 난이도별 내장 퍼즐 세트(02_data.md)를 순서대로 푼다.
7.2. 클리어한 퍼즐과 퍼즐별 최고 기록(최소 사용 수)을 저장한다.
7.3. 클리어 시 결과(사용 수 / 최소 수 / 최고 기록) 표시 후 다음 퍼즐로. 마지막 퍼즐을 클리어하면 완주 축하를 보여주고, 오버레이 버튼은 "처음부터 다시"(첫 퍼즐로)가 된다.
7.5. 상점: 상단 🛍 버튼으로 연다. 모은 골드로 주인공 토끼 색 스킨, 보드 테마(마을 길 색), 토끼 머리 장식(액세서리)을 구매(해금)하고 장착한다. 골드가 모자라면 구매되지 않는다. 장착한 보드 테마는 바닥·격자선·출구 길 색에, 액세서리는 토끼 머리에 즉시 반영된다.
7.4. 리셋 버튼으로 현재 퍼즐을 처음 상태로 되돌린다. 되돌리기(undo)로 한 수 전으로.
7.6. 힌트: 하단 💡 힌트 버튼으로 골드를 조금 써서 "지금 움직이면 좋은 동물 한 마리와 방향"을 알려준다. 솔버(BFS 최단 해)가 계산한 최적의 다음 한 수에 해당하는 차를 잠깐 강조하고 그 방향으로 살짝 움직여 보여준다(자동으로 옮겨 주지는 않는다 - 직접 민다). 골드가 모자라면 쓸 수 없다(골드 표시 흔들림). 이미 클리어했거나 더 풀 수가 없으면 동작하지 않는다.
7.7. 진행 맵: 상단 🗺 버튼으로 연다. 전체 퍼즐을 난이도별(입문/쉬움/보통/도전)로 묶어 번호 칩 격자로 보여준다. 각 칩에 퍼즐 번호와 받은 별 수(⭐)를 표시하고, 클리어한 퍼즐과 지금 보고 있는 퍼즐을 색으로 구분한다. 맨 위에 전체 진척(클리어한 퍼즐 수 / 모은 별 수)을 요약한다. 칩을 누르면 그 퍼즐로 바로 이동한다(이전·다음 버튼 없이 점프).

## 8. 화면 흐름

8.1. 상단 바: 홈(←) / 🛍 상점 / 🗺 진행 맵 / 🔊 음소거 / 퍼즐 번호·난이도 / 🪙 골드 / ⏱ 남은 시간 / 사용 수.
8.2. 중앙: 6x6 보드 + 차들 + 오른쪽 출구 표시.
8.3. 하단: 되돌리기 / 리셋 / 💡 힌트 / 이전·다음 퍼즐. (클리어 시) 다음 퍼즐 오버레이.
8.4. 클리어 시: 토끼 퇴장 애니메이션 + 축하 파티클 후 보드 위에 결과 오버레이(팝 등장).

## 9. 모바일

9.1. 모바일 지원(터치 드래그). 보드는 화면 너비에 맞춰 정사각형으로 반응형.
9.2. Pointer Events로 터치 / 마우스 단일 코드 경로.

## 10. 사운드

10.1. 짧고 귀여운 효과음으로 손맛과 성취감을 더한다. 외부 음원 파일 없이 Web Audio로 코드 합성한다(자산 0, vanilla).
10.2. 효과음 종류: 차 이동(톡), 클리어(밝은 상승 멜로디), 힌트(부드러운 띵), 상점 구매·장착(반짝), 거부=골드 부족(낮은 부저).
10.3. 음소거: 상단 🔊 / 🔇 버튼으로 켜고 끈다. 설정은 저장되어 다음에도 유지된다(02_data §4 `muted`). 기본은 소리 켬.
10.4. 브라우저 자동재생 정책상 첫 사용자 입력(탭/클릭) 후에 소리가 난다. 소리를 못 내도 게임 진행에는 영향 없다.
